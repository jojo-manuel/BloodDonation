# Blood Donation API

Node.js/Express + MongoDB backend for the Blood Donation app.

## Setup

1. Install deps

```
npm i
```

2. Copy env

```
cp .env.example .env
```

3. Run dev

```
npm run dev
```

## Routes

- POST /api/auth/register
- POST /api/auth/login
- POST /api/auth/refresh
- POST /api/auth/logout
- GET  /api/health
- GET  /api/users/me
- PATCH /api/users/me
- POST /api/donors/register
- GET  /api/donors/search
- GET  /api/donors/:id
